# Markdown Syntax

## create horizontal line (three hyphens)
---

## normal text (no syntax needed)
this is normal text

---

## bold/strong text 
**this is bold text**

---

## italic text
*this is italic text*

---

## strike off text
~this is not needed~

---


## sub-script
H<sub>2</sub>O

---


## super-script
X<sup>2</sup>

---


## blockquote text
>this is block quote

---
